# 移动端优化

在移动端，搜索框在获得焦点时会放大，并且在失去焦点后可以左右滚动，这可以通过设置元来优化。

// .vuepress/config.js
module.exports = {
  head: [
    ['meta', { name: 'viewport', content: 'width=device-width,initial-scale=1,user-scalable=no' }]
  ]
}

# 华为文案

首页可以显示 “华为” 文案，需要以下配置。

// .vuepress/config.js
module.exports = {
  themeConfig: {
    huawei: true
  }
}

# Logo

导航栏左侧可以显示logo, 需要以下配置。

// .vuepress/config.js
module.exports = {
  themeConfig: {
    logo: '/head.png'
  }
}
